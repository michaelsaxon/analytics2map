analytics2map

analytics2map is a static SVG renderer that turns Google Analytics visit telemetry, plus historical Clustrmaps exports, into self-hosted visitor maps. The pipeline pulls incremental GA4 data, merges in CSV/Text snapshots, stores everything in SQLite, and renders multi-scale SVGs behind a CLI. The stack is pure Python (Typer, Cartopy, svgwrite), producing two sizes by default: a compact 800×400 variant and a large 1920×960 version.

Each run emits two SVG files. The smaller render is easy to embed inline, while the larger map is tuned for high-resolution displays. Both share the same aggregated data and styling, and can be regenerated anytime after new ingests.

Source code is available in my [analytics2map] repository on GitHub.
