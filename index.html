analytics2map

analytics2map is a simple static site visitor map renderer by Michael Saxon. It turns Google Analytics visit telemetry and historical Clustrmaps exports into simple static visitor maps. The pipeline periodically pulls incremental GA4 data and puts it in SQLite to renders multi-scale SVGs. The stack is pure Python (Typer, Cartopy, svgwrite) and runs in a simple CLI. a compact 800×400 variant and a large 1920×960 version.

Each run emits two SVG files, an 800x400 one intended for embedding in a footer, and a higher-resolution one to be displayed full-width.

Every midnight, a Github actions workflow pulls the latest visit data from Google Analytics and updates the maps. The entire repo is also hosted on Github pages, so on my main website all I have to do is point to the output URL and as soon as the map gets updated, the changes are visible on my site.

Source code is available in my [analytics2map] repository on GitHub.
